# डेटा प्रविष्टि केंद्र

हिंदी में तैयार किया गया यह वेब ऐप टीमों को कर्मचारियों या ग्राहकों का डेटा तेज़ी से एकत्र और प्रबंधित करने में मदद करता है।

## विशेषताएँ

- अनिवार्य फ़ील्ड का सत्यापन (नाम, ईमेल, मोबाइल, विभाग, शहर, राज्य, पिनकोड)
- रिकॉर्ड जोड़ें, संपादित करें और हटाएं
- शीर्ष विभाग और कुल रिकॉर्ड का त्वरित सारांश
- CSV प्रारूप में डेटा डाउनलोड
- सभी पाठ हिंदी में ताकि स्थानीय टीम आसानी से उपयोग कर सके

## विकास

```bash
npm install
npm run dev
```

एप्लिकेशन `http://localhost:3000` पर उपलब्ध होगा।

## बिल्ड

```bash
npm run build
npm run start
```
